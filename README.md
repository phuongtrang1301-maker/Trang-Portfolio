# Portfolio Starter

A clean, responsive portfolio template you can customize for job applications.

## Run locally

1. Open `/Users/tranghoang/Documents/New project/index.html` in your browser.
2. Or run a local server from this folder (example: `python3 -m http.server 8000`) and visit `http://localhost:8000`.

## Customize before applying

Edit these files:

- `/Users/tranghoang/Documents/New project/index.html`
  - Replace `YOUR NAME`, bio text, email, LinkedIn, GitHub, and resume link.
  - Update section copy to match your experience.
- `/Users/tranghoang/Documents/New project/script.js`
  - Replace sample projects in the `projects` array.
  - Add real `liveUrl` and `githubUrl` links.
- `/Users/tranghoang/Documents/New project/styles.css`
  - Adjust colors in `:root`.
  - Tune spacing/typography if needed.

## Suggested project card format

For each project, include:

- What problem you solved
- What you built
- Measurable impact (`+X%`, `-Y mins`, `$Z saved`)
- Tech stack
